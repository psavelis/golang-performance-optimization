# Observability MCPs

These MCPs focus on fast, reliable interrogation of metrics, logs, and continuous profiles.

Design Tenets:
- Deterministic defaults, tunable via query parameters
- Avoid over-fetching: narrow time windows & label selectors
- Emit structured diagnostics + human summary

Included MCPs:
1. Prometheus Query Assistant
2. Prometheus Alert Explainer
3. Pyroscope Profile Explorer
4. Pyroscope Diff Analyzer
5. Grafana Dashboard Author
6. Log Aggregation Slicer

Each entry details contracts and examples.
